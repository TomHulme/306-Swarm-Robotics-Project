1. Install Git:

   Assuming you are using Ubuntu, open terminal.
	$ apt-get install git

   Follow the prompts. Simple

2. Set configs:
   
   Git needs to know things to keep track of your commits. So, in Terminal, type
	$ git config --global user.name "[your name]"
   
	$ git config --global user.email "[the email you used for github]"
	
	$ git config --global credential.helper cache	


3. Initialise local repository:

   First off, in the file explorer (or terminal, if you want), create a folder that will hold the repository. Then, in Terminal, 
    navigate to the folder, type
	$ git init 

   This will create all the git related files
